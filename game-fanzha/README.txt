反诈先锋游戏安装包说明
==========================

概述
----
这是一个可以在Windows 7及以上系统运行的《反诈先锋》游戏安装包。

系统要求
--------
- 操作系统：Windows 7/8/10/11
- 需要安装Python 3.6或更高版本
- 至少100MB可用磁盘空间

安装说明
--------
1. 运行"反诈先锋安装程序.exe"进行安装
2. 按照安装向导的提示完成安装
3. 安装完成后，可以从开始菜单或桌面快捷方式启动游戏

手动安装（备用方案）
------------------
如果安装程序无法正常工作，可以手动安装：
1. 将game-fanzha文件夹复制到任意位置
2. 双击运行start-game.bat启动游戏
3. 在浏览器中访问 http://localhost:8000/desktop-version.html

游戏功能
--------
- 防诈骗知识学习
- 多种诈骗场景模拟
- 实时答题统计
- 成就系统
- 学习进度跟踪

技术支持
--------
如遇到问题，请检查：
1. 是否已安装Python
2. 防火墙是否阻止了本地服务器
3. 端口8000是否被其他程序占用

卸载说明
--------
通过控制面板的"程序和功能"卸载，或直接删除安装文件夹。
